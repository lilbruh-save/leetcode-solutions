> [242. Valid Anagram](https://leetcode.com/problems/valid-anagram/)
> **Solution count:** `1 / 3`

#### 🧠 Условие
Для двух строк `s` и `t` верни `true`, если `t` является **анаграммой** `s`, и `false` в противном случае.

Анаграмма - это когда в строках **одни и те же буквы** и **в том же количестве**, только порядок может быть другим.

#### 🧪 Примеры

- Input: `s = "anagram", t = "nagaram"` -> Output: `true`
- Input: `s = "rat", t = "car"` -> Output: `false`

#### ✅ Решения

##### 💡 Hint

**Идея:**  
Используем массив из 26 элементов, чтобы хранить баланс букв.
- для `s` увеличиваем счётчик
- для `t` уменьшаем счётчик
Если в конце весь массив состоит из нулей, значит буквы и их количество совпадают.

**Шаги:**

1. Сравнить длины `s` и `t`
2. Создать `count [26]int`
3. Пройтись по строкам
4. Для каждой буквы:
   - Увеличить счётчик для буквы из `s`
   - Уменьшить счётчик для буквы из `t`
1. Проверить массив `count`
2. Если где-то не `0` - вернуть `false`
3. Иначе вернуть `true`
#####  Code:
```go
func isAnagram(s string, t string) bool {
    if len(s) != len(t) {
        return false
    }

    var count [26]int

    for i := 0; i < len(s); i++ {
        count[s[i]-'a']++
        count[t[i]-'a']--
    }

    for _, v := range count {
        if v != 0 {
            return false
        }
    }
    return true
}
```

- **Время:** `O(n)`  
- **Память:** `O(1)`
