> [1. Two Sum](https://leetcode.com/problems/two-sum/)
> **Solution count:** `1 / 2`

#### 🧠 Условие
Дан массив целых чисел `nums` и целое число `target`.
Верни **индексы двух чисел**, сумма которых равна `target`.

Гарантируется, что:
- решение ровно одно
- один и тот же элемент нельзя использовать дважды
- ответ можно вернуть в любом порядке

#### 🧪 Примеры

- Input: `nums = [2,7,11,15], target = 9` -> Output: `[0,1]`
- Input: `nums = [3,2,4], target = 6` -> Output: `[1,2]`
- Input: `nums = [3,3], target = 6` -> Output: `[0,1]`

#### ✅ Решения

##### 💡 Hint

**Идея:**  
Используем `hash map` через `map[int]int`.  
Пока идём по массиву, для каждого числа ищем его дополнение до `target`.  
Если дополнение уже встречалось раньше - сразу возвращаем два индекса.  
Если нет - сохраняем текущее число и его индекс.

**Шаги:**

1. Создать `map`: `число -> индекс`
2. Пройтись по массиву `nums`
3. Для каждого `num`:
   - Посчитать `need = target - num`
   - Если `need` уже есть в `map` - вернуть два индекса
   - Иначе добавить `num` и его индекс в `map`
4. Если цикл закончился - вернуть `nil`

#### 🔹 Решение 1
```go
func twoSum(nums []int, target int) []int {
    m := make(map[int]int)

    for i, num := range nums {
        need := target - num
        if j, ok := m[need]; ok {
            return []int{j, i}
        }
        m[num] = i
    }
    return nil
}
```

- **Время:** `O(n)`  
- **Память:** `O(n)`
