> [217. Contains Duplicate](https://leetcode.com/problems/contains-duplicate/) 
> **Solution count:** `1 / 3`

#### 🧠 Условие
Дан массив целых чисел `nums`.  
Верни `true`, если **хотя бы одно число встречается 2 раза и больше**.  
Иначе верни `false`.

#### 🧪 Примеры

- Input: `[1,2,3,1]` -> Output: `true`
- Input: `[1,2,3,4]` -> Output: `false`
- Input: `[1,1,1,3,3,4,3,2,4,2]` -> Output: `true`

#### ✅ Решения

##### 💡 Hint

**Идея:**  
Используем `set` через `map[int]struct{}`.  
Пока идём по массиву, проверяем, встречалось ли текущее число раньше.  
Если да - сразу возвращаем `true`.  
Если нет - добавляем число в `set`.

**Шаги:**

1. Создать `set`: `число -> struct{}`
2. Пройтись по массиву `nums`
3. Для каждого `num`:
   - Если `num` уже есть в `set` - вернуть `true`
   - Иначе добавить `num` в `set`
4. Если цикл закончился и повторов не было — вернуть `false`

##### Code:
```go
func containsDuplicate(nums []int) bool {
    set := make(map[int]struct{})
    for _, num := range nums {
        if _, ok := set[num]; ok {
            return true
        }
        set[num] = struct{}{}
    }
    return false
}
```

- **Время:** `O(n)`  
- **Память:** `O(n)`
